A demo of an interactive map on zone scenarios for Across Lines.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

Navigate to the home folder of this app in your console, and then run:

- `npm run start` to run a local version at localhost:3000 for local development.
- `npm run build` to create an iframe-embeddable, production-ready version of the app in the "build" subfolder.
